# Residential Market Intelligence Dashboard

## Project Overview

This project analyzes residential real estate sales in Chicago using MLS transaction data. 
The goal is to explore housing market trends, property characteristics, pricing patterns, and 
neighborhood-level differences through data cleaning, SQL analysis, and interactive dashboards.

## Tools

- R
- SQL
- Power BI
- GitHub

## Current Status

Data cleaning and exploration in progress.

## Dataset

The project currently uses single-family home sales data from Humboldt Park, Chicago. Additional 
neighborhoods may be added in future analyses.
